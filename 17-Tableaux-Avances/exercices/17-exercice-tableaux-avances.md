# Exercice 17 — Tableaux avancés

## Objectif
Utiliser le masquage, `where` et `forall`.

## Instructions
1. Créez un tableau 5x5 d'entiers aléatoires (0-99).
2. Utilisez `where` pour remplacer les >50 par 0.
3. Utilisez `forall` pour mettre les indices pairs à -1.

## Code de départ
```fortran
program tableaux_avances
  implicit none
  integer, dimension(5,5) :: t
  integer :: i, j
  ! À compléter
end program tableaux_avances
```
