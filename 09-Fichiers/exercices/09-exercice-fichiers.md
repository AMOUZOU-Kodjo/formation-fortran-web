# Exercice 09 — Lecture et écriture de fichiers

## Objectif
Lire et écrire dans un fichier texte.

## Instructions
1. Écrivez 10 nombres dans `donnees.txt`.
2. Relisez le fichier et affichez les nombres.
3. Ajoutez la somme à la fin du fichier.

## Code de départ
```fortran
program fichiers
  implicit none
  ! À compléter
end program fichiers
```
