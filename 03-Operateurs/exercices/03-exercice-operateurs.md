# Exercice 03 — Opérateurs arithmétiques et logiques

## Objectif
Utiliser les opérateurs `+`, `-`, `*`, `/`, `**` et les opérateurs logiques.

## Instructions
1. Calculez `(5 + 3) * 2**3 / 4`.
2. Testez les opérateurs de comparaison (`<`, `>`, `==`, `/=`).
3. Affichez les résultats.

## Code de départ
```fortran
program operateurs
  implicit none
  ! À compléter
end program operateurs
```
