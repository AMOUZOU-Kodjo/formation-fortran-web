# Exercice 01 — Premier programme Fortran

## Objectif
Écrire, compiler et exécuter un premier programme Fortran 95.

## Instructions
1. Créez un fichier `premier.f95`.
2. Écrivez un programme qui affiche `"Bonjour, monde !"` et votre prénom.
3. Compilez avec `gfortran -Wall -std=f95 -o premier premier.f95`.
4. Exécutez le programme.

## Code de départ
```fortran
program premier
  implicit none
  ! À compléter
end program premier
```

## Vérification
- La compilation ne produit ni erreur ni avertissement.
- L'exécution affiche le message attendu.
