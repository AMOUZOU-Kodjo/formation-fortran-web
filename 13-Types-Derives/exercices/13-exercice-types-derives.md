# Exercice 13 — Types dérivés

## Objectif
Définir et utiliser un type dérivé.

## Instructions
1. Définissez un type `Point` avec x, y (reels).
2. Créez deux points et calculez la distance entre eux.

## Code de départ
```fortran
program types_derives
  implicit none
  type :: Point
    real :: x, y
  end type Point
  ! À compléter
end program types_derives
```
