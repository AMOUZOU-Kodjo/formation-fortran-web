# Exercice 08 — Entrées/Sorties formatées

## Objectif
Utiliser les formats d'affichage et les entrées clavier.

## Instructions
1. Lisez un nom et un âge.
2. Affichez avec un format : "Nom: [nom], Age: [age]".

program io
  implicit none

  ! À compléter
