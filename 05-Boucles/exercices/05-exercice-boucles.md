# Exercice 05 — Boucles

## Objectif
Utiliser la boucle `do` pour générer une table de multiplication.

## Instructions
1. Lisez un entier N au clavier.
2. Affichez la table de multiplication de N de 1 à 10.

## Code de départ
```fortran
program table
  implicit none
  integer :: n, i
  write(*,*) "Entrez un nombre :"
  read(*,*) n
  ! À compléter
end program table
```
